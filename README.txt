To install this plugin, just put it in your vendor/plugins/ directory and restart Redmine.

Please note it requires "ultraviolet" gem, you could install it with "gem install ultraviolet".
This installation needs oniguruma lib, which can be installed on Ubuntu or Debian systems with
the "libonig-dev" package. For other systems, refer to http://ultraviolet.rubyforge.org/ and
http://textpow.rubyforge.org/.
